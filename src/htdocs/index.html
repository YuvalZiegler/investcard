The process

As a first step I created a quick prototype for each of the states using Balsamiq and took some notes on the UX and the UI.

I picked the Flux application architecture along with React for the purpose of this demo. Flux and React are used on large scale application at facebook. Flux complements React's composable view components by utilizing a unidirectional data flow, it is more of a pattern rather than a formal framework, and I think it's unidirectional nature makes it easier to reason about compared to traditional data binding. React uses a virtual DOM diff implementation for ultra-high performance. It can also render on the server which allows it to be Search engine optimized.

I used Gulp with Browserify to preprocess and bundle the assets.
